### 大學專題 *2013 ~ 2014*
> 指導教授: 王家慶 教授 
- 利用多層架構之音樂情緒辨識系統: 此系統利用兩層架構去進行音樂情緒辨識，分別為稀疏代表 (sparse representation)和支援向量機 (Support Vector Machine)
- 辨識系統是利用 Matlab 實作，另外也使用 Java 去實作應用介面讓使用者選擇聆聽情緒接近的音樂
