# 專案

### Unreal 個人遊戲開發 _2023 Mar ~ 2024 Feb_

- 練習開發 ARPG 線上遊戲，包含遊戲客戶端、啟動器、網頁與後端伺服器
- 使用 Unreal (C++ & Blueprint) 實作客戶端與 Dedicated Server，GAS 實作技能系統
- 後端使用 Golang + Agones + Redis + Spanner，gRPC with SSL，架設於 GKE

### Unity 個人遊戲開發 _2020 Aug ~ 2021 Apr_

- 撰寫多人派對遊戲，學習 DOTS 架構、Unity New Input System、Addressable Asset System
